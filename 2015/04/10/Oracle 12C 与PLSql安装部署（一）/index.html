Oracle 12C 与PLSql安装部署（一）

By KNNA

发表于 2015-04-10

文章目录
  • 准备下载Oracle 12C windows-64bit版的数据库安装包。

  • 官方下载地址：http://www.oracle.com/technetwork/database/enterprise-edition/downloads/index.html打开页面后点击 Accept License Agreement ，即可点击 File 1 ， File 2 进行下载了。选择带有 Windows 的版本。
  • 将下载好的两个文件解压到同一个文件夹（database）中，使其合并为一个文件夹。
  • 打开database文件夹中“setup.exe”，开始安装程序，勾掉复选框，点击下一步。

  • 点击是，点击下一步。

  • 选择跳过软件更新，点击下一步。


  • 选择创建和配置数据库，点击下一步。


  • 选择服务器类，点击下一步。


  • 选择单实例数据库安装，点击下一步。


  • 选择高级安装，点击下一步。


  • 选择语音，默认就好，直接点击下一步。


  • 选择版本，这里选择的是企业版，点击下一步。


  • 选择使用windows内置账户，点击下一步。


  • 点击是继续。


  • 设置Oracle数据库软件的安装目录和Oracle数据库配置文件相关的基目录，根据需要设置。点击下一步。


  • 选择一般用途/事物处理，点击下一步。


  • 默认就好，我这里出现错误提示是因为之前已经安装过一次，所以提示标识符（orcl）已存在，新安装直接点击下一步。


  • 默认就好，点击下一步。


  • 设置数据库文件的目录，设置好后点击下一步。


  • 不勾选注册，点击下一步。


  • 我这里没有启用恢复，直接点击下一步。


  • 设置每个账户的口令，这里我同意设置了一个相同的口令，当然也可以单独为每个账户设置指令，记得记录下每个账户所设置的密码，点击下一步。


  • 若密码强度不高，会提示不符合标准，点击是继续。


  • 到这里就列出了我们之前做的所有设置清单。没有问题就点击安装。



  • 安装完成，下一节（）介绍如何安装PL/Sql并连接到Oracle。

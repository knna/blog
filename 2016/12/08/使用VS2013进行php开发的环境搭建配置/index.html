使用VS2013进行php开发的环境搭建配置

发表于 2016-12-08

下载准备

  • 首先下载XAMPP套件，包含了Apache、MySQL、phpmyadmin等组件。选择个最新版本下载即可，我用的7.0.1。下载地址
  • 下载VS 2013插件：PHP Tools for VS2013，下载地址

安装XAMPP

打开下载的套件进行安装，安装好后，启动XAMPP Control Panel，可以看到Service列有四个项目是红叉，我们分别点击前三个进行安装，安装完成后Service列对应的项目变为绿勾。
这时可以依次点击已安装组件Actions列的Start按钮启动组件。
如果Apache组件启用失败，可能是因为80端口被占用的原因，找到安装目录下的 ～\xampp\apache\conf\httpd.conf ,把所有默认端口80全部改为8080，保存后重新启动即可。
启动完成如图：

启动完成后，PHP环境就搭建好了，打开 http://localhost:8080/ 可以看到欢迎界面。
我们可以写个简单的php页面测试：

保存为 helloworld.php并拷贝到～\xampp\htdocs，启动浏览器输入 http://localhost:8080/helloworld.php ，没有问题的话看到我们的页面已经被解析出来了。

安装PHP Tools for VS2013

因为一直用VS做开发，已经习惯了VS的开发方式，所以网上百度了一下是否有PHP在VS下开发的插件，一看果然有，不过是收费的，上面提供的下载是破解版的。
解压安装包，打开PhpTools.1.15.5962.msi 开始安装，安装完成后，拷贝Crack文件夹下面的DLL文件到 C:\Users\administrator\AppData\Local\Microsoft\VisualStudio\12\Extensions\DEVSENSE\PHP Tools for Visual Studio 2013 1.15.5962\ 中。
破解完成。可以打开VS新建项目，看到有PHP的项目可以创建了。

在VS中点击调试的时候，第一次会弹出对话框，提示没有调试组件，我们选择第一项下载安装并耐心等待安装结束，然后重新打开调试器即可。
